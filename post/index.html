033 ways to join

三种参展方式，对号入座

不管你能不能亲自到场、想投入多少，都有一种适合你的方式。下面逐个说明它适合谁、流程怎么走、交付什么、最后你能得到什么。

1
Method 01 · 全程陪同

商家自己出摊

你亲自到场，我们全程陪同保障
适合谁Who
想亲自到场、亲眼看市场反应、愿意自己运营的品牌；通常处于出海探索期，创始人或团队能出差。
流程怎么走How
  1. 机酒安排：可包或不包（按需选择）
  2. 行程前准备：办当地电话卡、行程攻略等
  3. 接机 + 展会提前踩点：租商务车，开展前先看场地、看对手
  4. 布展：可选「设计 / 设计+布置 / 仅出人工」三档
  5. 现场售卖 + 消费者调研（见下方亮点）
  6. 【可选】展会后本地化：本地走访调研、线下商圈考察、消费习惯洞察、品牌参访（可能促成联名）
★ 招牌动作 · 扮演"调研员"

我们不只是收款、拍照。现场会有人扮演调研员，记录消费者在摊位前的停留时长、触摸材质时的真实反应、对价格的真实吐槽，并收集 50–100 份结构化问卷，展后交付一份定性分析报告——这些是你坐在国内永远拿不到的一手洞察。

交付清单Deliverables
  • 定性消费者调研报告（停留 / 反应 / 价格反馈）
  • 50–100 份问卷原始数据
  • 现场照片素材
  • （可选）本地化走访洞察与联名线索
你能得到Outcome
亲历真实市场 + 一份扎实的定性消费者调研报告 + 可落地的本地化线索，回国就能拍板下一步。
2
Method 02 · 全包执行

代出摊 / 合并出摊

你不用到场，我们替你把摊出好
适合谁Who
人不能到场、想低成本试水的品牌。可与其他 3–4 家混合品类合并一个摊位分摊成本，也可单独包摊。
流程怎么走How
  1. 你提供：产品 + 物料 + 介绍手册 / 产品价格表 / 优惠最低价
  2. 产品了解 meeting：我们吃透你的产品卖点
  3. 培训销售人员：让现场兼职能讲清你的故事
  4. 出摊 + 实时跟进：开展期间随时同步售卖情况
  5. 代收款：手工记录每一笔售卖情况
  6. 市场报告：销售情况 + 品类推荐 + 市场进入建议
  7. 产品返还 / 销售额返回：未售产品退回，销售额结算给你
交付清单Deliverables
  • 市场报告（销售情况 + 品类推荐 + 市场进入建议）
  • 销售对账记录（手工记录 + 拍照留证）
  • 销售额返回 + 未售产品返还
  • 现场照片素材
你能得到Outcome
人不到场，也能完成一次真实市场测试，拿到市场报告与透明结算，零差旅成本试水东南亚。
费用怎么算：参展费、物流费、翻译、兼职销售、我们的服务费——可选 ① 合并摊位（3–4 家分摊，更省）或 ② 不合并（独立包摊，更专属）两档。具体价格见第 05 节对比表。
3
Method 03 · 按结果分成

寄卖 · 分利润

我们精选品牌，几乎零前期投入
适合谁Who
我们自己看好、确定有市场的品牌；以及想几乎零前期投入、按结果付费的商家。
流程怎么走How
  1. 我们选品：只挑我们判断有市场的产品
  2. 寄卖上架：把你的产品放进我们的渠道 / 摊位
  3. 销售：由我们负责卖
  4. 按利润分成结算：卖出后按约定比例分成
交付清单Deliverables
  • 销售数据 + 利润分成结算单
  • 市场反馈摘要
  • 长期合作 / 复购建议
你能得到Outcome
近乎零前期投入就进入东南亚市场，拿到真实销量验证，跑通后还有长期合作的可能。
04After the fair

展会之后：把一次展会变成长期生意

额外收费以下均为可选的增值服务，单独计费，可按需组合在任意一种参展方式之上。
01

深度市场报告

在基础数据之上，提供更系统的市场分析、品类与定价建议、进入路径规划。

02

线上独立站搭建 + 推广

帮你搭建出海独立站并做本地化推广，把线下验证过的产品搬到线上持续卖。

03

寄卖 / 代寄卖找合作

对接当地渠道与合作方，帮你把货持续铺出去，不止于一次展会。

04

本地化转化

把展会期间的本地洞察，转化成真正落地的合作机会：

  • 当地品牌联名合作
  • 当地投资机会对接
  • 线上出海渠道（独立站 + 本地化营销）

06Trust & how we pick

我们怎么选展会，以及怎么让你放心

选展会的标准

  • 人流量与到场客群是否匹配你的目标人群
  • 展会品类调性与你的产品是否契合
  • 参展费用与预期回报是否划算
  • 当地市场对该品类的接受度

透明结算 · 风险保障

  • 每一笔售卖手工记录 + 拍照留证
  • 未售出产品如数返还
  • 销售额清晰对账后返回
  • 过程实时同步，不做黑箱
07FAQ

常见问题

语言不通怎么办？

方式二/三由我们配备会当地语言的销售与翻译，方式一现场也有翻译支持，沟通无障碍。

产品怎么过去、清关复杂吗？

物流与清关由我们协助安排，费用包含在参展成本里（详见报价）。

卖不完的货怎么处理？

未售出产品如数返还；也可选择转入"寄卖/找合作"在当地继续销售。

钱什么时候结给我？

展会结束对账后按约定周期结算。

想让你的产品，先去东南亚试一试？

告诉我们你的产品和目标，我们帮你挑一场最合适的展会，定一套最划算的方式。
